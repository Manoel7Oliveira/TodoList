# Todo List

Aplicação de lista de tarefas desenvolvida com HTML, CSS e JavaScript.

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript 

## 📌 Funcionalidades

- Adicionar tarefas
- Editar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Manipulação dinâmica da interface

## 💡 Conceitos praticados

- Manipulação do DOM
- Event Listeners
- Delegação de eventos
- Criação dinâmica de elementos
- Organização do código em funções
- Manipulação de classes CSS

## 📚 Aprendizados

Este projeto foi desenvolvido com o objetivo de praticar manipulação do DOM, eventos, funções e lógica de programação utilizando JavaScript.
